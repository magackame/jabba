
# jabba

WebUI for scraping domain names from [porkbun](https://porkbun.com).

## features

- scrape domains
  - name
  - price
  - renewal price
  - date of the latest entry on [Wayback Machine](https://web.archive.org/)
- excel export of scrape results

## dependencies

- JDK 21
- maven
- chromedriver
- Google Chrome

## run

`./run.sh` or `run.cmd`
